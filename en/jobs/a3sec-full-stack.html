Jobs

Full stack developer

Necesitamos incorporar en A3Sec Madrid desarrolladores con entre 2 y 4 años de experiencia tanto en backend como en frontend para desarrollar junto a un equipo de I+D+i una herramienta de monitorización de última generación.

Los candidatos deberán dominar las siguientes tecnologías:

Back:

  • Python + django
  • Control de versiones con Git
  • Administración básica de sistemas Linux (principalmente debian y ubuntu)
  • Desarrollo y uso de APIs REST

Front:

  • HTML5 + CSS3
  • javascript + jquery y algún framework javascript preferiblemente react.js

Se valorará muy positivamente el conocimiento en cualquiera de las siguientes tecnologías:

  • Scripting Linux (bash, python, perl, ruby)
  • Motores de bases de datos: MySQL, MongoDB, Redis, Casandra
  • Despliegue de configuraciones (salt, puppet, ansible, chef)
  • Servidores web (apache, nginx, gunicorn, etc.)
  • Contenedores
  • Stack ELK (Elastic)
  • Preprocesadores CSS (SASS, LESS)
  • Empaquetado de software (deb y/o rpm)

Salario competitivo dependiendo de experiencia y valía, beneficios sociales, horario flexible, teletrabajo y toda la Coca-Cola que puedas tomar.

Envía tu CV a rrhh@a3sec.com con la Referencia Full Stack Developer.

Te esperamos!!!
